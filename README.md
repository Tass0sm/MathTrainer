# mental-math-mode

## Overview

A simple Emacs major-mode for completing small math problems.

## Default Key-binds

| Key-bind    | Command             |
| ----------- | ------------------- |
| `C-c C-n`   | insert-problem      |
| `C-c C-r`   | set-operand-limit   |
| `Return`    | complete-problem    |
